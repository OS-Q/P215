.. _modbus_std:

standard
~~~~~~~~~~~~~~~~~

* 双串口，至少一路支持RS485通信
* 串口波特率2400/4800/9600/19200/38400/57600/115200
* 至少一个数字输出IO，切换频率≥ 1KHz
* 至少一个数字输入IO，采样频率≥ 1KHz
